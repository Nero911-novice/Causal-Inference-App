# Causal Inference App

Streamlit-приложение для оценки инкрементального эффекта маркетингового воздействия через аналитическую триангуляцию.  
Приложение строит несколько контрфактуальных baseline-сценариев и сравнивает их с фактическим результатом в тестовом периоде.

## Что делает приложение

Приложение:

- принимает CSV или Excel с временным рядом;
- рассчитывает месячную динамику (`MoM`) для целевой и контрольной групп;
- строит три независимые оценки baseline;
- считает чистый эффект в процентных пунктах и в абсолютных значениях;
- показывает итоговые таблицы, график и проверки устойчивости;
- формирует краткую интерпретацию силы сигнала и устойчивости эффекта.

## Входные данные

Ожидаются три колонки:

- `Date` — дата;
- `Target_Group` — абсолютные значения целевой группы;
- `Control_Group` — абсолютные значения контрольной группы или рыночного ориентира.

После загрузки приложение:

1. приводит даты к месячному уровню;
2. агрегирует повторяющиеся строки внутри месяца;
3. рассчитывает месячную динамику:

```text
MoM = (V_t - V_t-1) / V_t-1
```

## Периоды анализа

В интерфейсе выбираются два окна:

- `Historical Period` — исторический период для оценки моделей;
- `Test Period` — период воздействия, для которого считается эффект.

## Используемые методы

### 1. Регрессия чувствительности к рынку

На историческом периоде оценивается связь:

```text
Target_MoM = beta * Control_MoM
```

На тестовом периоде baseline строится через фактическую рыночную динамику и коэффициент `beta`.

### 2. CV Leverage

Оценивается относительная волатильность целевой и контрольной групп через коэффициент вариации:

```text
CV = std / mean
L = CV_control / CV_target
```

Прогноз строится как рыночная динамика, скорректированная на коэффициент `L`.

### 3. Сезонная модель YoY

Для каждого тестового месяца используется средний исторический `Target_MoM` для того же календарного месяца.

## Расчет эффекта

Для каждой модели приложение сравнивает:

- фактическую динамику `Target_Group`;
- прогнозную динамику без воздействия.

Чистый эффект считается как:

```text
Impact_pp = Fact_MoM - Forecast_MoM
```

Далее baseline переводится в абсолютную контрфактуальную траекторию, и экономический эффект рассчитывается как разница между фактом и baseline в абсолютных значениях.

## Проверки устойчивости

В приложении реализованы четыре блока проверки устойчивости:

- `Placebo-анализ` — псевдо-тестовые окна внутри historical периода;
- `Leave-one-year-out` — исключение одного года из historical периода;
- `Leave-one-month-out` — исключение одного месяца из historical периода;
- `Границы Historical` — сдвиг начала и конца historical периода.

Эти проверки помогают понять:

- насколько результат выделяется на фоне ложных окон;
- зависит ли эффект от одного года или одного месяца;
- чувствителен ли вывод к выбору границ historical периода.

## Интерпретация результата

Приложение показывает:

- качество моделей;
- сводную таблицу триангуляции;
- устойчивость эффекта по стресс-сценариям;
- автоматическую текстовую интерпретацию для placebo и robustness-проверок.

Интерпретация строится как прикладная аналитическая оценка.  
Она не заменяет экспериментальный причинный дизайн и не является формальным статистическим доказательством причинности.

## Структура проекта

Текущая структура проекта:

```text
causal_inference_app.py   # Streamlit-интерфейс
data_prep.py              # загрузка, парсинг и подготовка данных
models.py                 # математическое ядро моделей
robustness.py             # проверки устойчивости
interpretation.py         # текстовая интерпретация результатов
reporting.py              # форматирование, таблицы, график, экспорт
requirements.txt
```

## Локальный запуск

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск приложения:

```bash
streamlit run causal_inference_app.py
```

## Развертывание

Приложение можно запускать локально или деплоить в Streamlit Community Cloud.  
Для облачного запуска достаточно указать репозиторий, ветку и файл входа `causal_inference_app.py`.

## Ограничения

Приложение подходит для прикладной оценки эффекта, но требует аккуратной интерпретации:

- контрольная группа должна быть содержательно связана с целевой;
- короткий historical период снижает устойчивость выводов;
- сильные структурные изменения бизнеса могут искажать baseline;
- эффект следует трактовать как оценку через набор контрфактуальных сценариев, а не как окончательное доказательство причинности.

## Назначение результата

Инструмент подходит для:

- внутренней маркетинговой аналитики;
- оценки инкрементального эффекта кампаний;
- сравнения нескольких baseline-сценариев;
- проверки устойчивости вывода при малом объеме данных.
